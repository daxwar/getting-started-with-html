How the Web Works


HTTP (Hypertext Transfer Protocol) is the set of rules for transferring files (text, graphic images, sound, video, and other multimedia files) on the World Wide Web. As soon as a Web user opens their Web browser, the user is indirectly making use of HTTP.

HTML Tag is a code that describes how a Web page is formatted. HTML tags are defined by the characters < & >.
html tag examples-
b renders text bold
em content-italic,Renders as emphasized text
a defines a hyperlink
br inserts a single line break
p tag defines a paragraph.
span tag is used to group inline-elements in a document
div tag defines a division or a section in an HTML document.
img tag defines an image in an HTML page.

HTML Attribute is a modifier of an HTML element
html attribute examples:
href
src
width
height

Inline vs Block-inline elements render inline with the HTML block elements create an invisible block around the content

HTML Element is an individual component of an HTML document or web page

Whitespace- blank space in an HTML page. Relevant tags are br, p, span,div
